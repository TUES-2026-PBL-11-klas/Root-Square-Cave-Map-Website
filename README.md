# Root-Square-Cave-Map-Website

## (BG)

### Проектът представлява уебсайт, който включва карта на пещерите в България. При избор на дадена пещера се показват координати, кратко описание, визуалицация и връзка за повече информация.
### Има меню за търсене и филтриране по град и/или община, както и система за проверка за валидност на профила.


## (EN)

### This project is about a website that includes a map of caves in Bulgaria. When a given cave is picked, coordinates, a vague description, visualization and a link for more information.
### Among the included features are a search menu (with a filter for a town/city and/or municipality), as well as a profile verification system.
